## Development

```bash
    # clone the project
    git clone https://github.com/LonelVino/club-chinois-home.git

    # enter the project directory
    cd club-chinois-home
    
    # install dependencies
    npm install
    //or 
    npm install --registry=https://registry.npm.taobao.org


    # develop locally
    npm run dev
```

## Build Deploy
